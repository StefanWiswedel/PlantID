# PlantID — Sjælland Field Guide

A web-based field guide to common plant species found on Sjælland, Denmark. Starting with 21 tree species, the app is designed to expand to shrubs, herbaceous plants, annuals, and more.

## Species Included (Trees)

### Native Broadleaves (13)
- European Beech (*Fagus sylvatica*)
- Pedunculate Oak (*Quercus robur*)
- European Ash (*Fraxinus excelsior*)
- Silver Birch (*Betula pendula*)
- Downy Birch (*Betula pubescens*)
- Common Alder (*Alnus glutinosa*)
- Small-leaved Lime (*Tilia cordata*)
- European Hornbeam (*Carpinus betulus*)
- Norway Maple (*Acer platanoides*)
- White Willow (*Salix alba*)
- Rowan (*Sorbus aucuparia*)
- Aspen (*Populus tremula*)
- Wild Cherry (*Prunus avium*)

### Introduced Broadleaves (1)
- Sycamore Maple (*Acer pseudoplatanus*)

### Conifers (7)
- Common Yew (*Taxus baccata*)
- Norway Spruce (*Picea abies*)
- Sitka Spruce (*Picea sitchensis*)
- Scots Pine (*Pinus sylvestris*)
- Japanese Larch (*Larix kaempferi*)
- Nordmann Fir (*Abies nordmanniana*)
- Douglas Fir (*Pseudotsuga menziesii*)

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS v4
- React Router
